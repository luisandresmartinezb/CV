# 💼 CV - Luis Andrés Martínez Berraquero

Currículum desarrollado en **HTML, CSS y JavaScript**, diseñado para ser visualizado en navegador y optimizado para impresión en formato A4.

---

## 🚀 Descripción

Este proyecto consiste en mi currículum web, donde se presenta mi experiencia profesional como desarrollador con enfoque Full Stack, incluyendo:

- Experiencia profesional en desarrollo y soporte técnico
- Tecnologías y herramientas utilizadas
- Formación académica
- Información de contacto

El diseño está pensado para ser limpio, profesional y fácilmente imprimible en una sola página.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Diseño responsive
- Optimización para impresión (A4)

## 🖥️ Cómo usarlo

1. Clona el repositorio:

```bash
git clone https://github.com/luisandresmartinezb/CV.git
```
2. Abre el archivo:
```
index.html
```
3. Para exportar a PDF:
   + Abre en Chrome o Edge
   + Pulsa Ctrl + P
   + Selecciona "Guardar como PDF"
   + Formato: A4
